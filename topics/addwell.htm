Add wells to Asset Model

 

 

 

 

Well can be added or deleted by clicking on the (Add /Remove wells) icon in the Network modelling interface.

 

 

 

Add Well

Wells can be added using the interface below. Existing but also future wells need to be added to be part of the Asset model.

The minimum required information are:

 - Well Name

 - Type of Well: Oil producer, Gas Producer, Water Injector or Gas injector. This is the type at start of forecast. The well can eventually be converted later in the forecast, see section Well conversion

 - the Node the well flows to (for producers). For injectors, Users need to input the name of the Node where injection fluid comes from.

 - the name of reservoir

 - the project it is part of. The project are used to group production by project when visualizing forecast results.

 

 

Once a well has been added to the Asset model, its decline curves needs to be defined using the DCA interface (see Decline curves: Production and Injection potentials )

For future well, the starting date will be specified by creating an Opportunity (see Opportunities) and placed in the drilling sequence (see Sequence of events )
