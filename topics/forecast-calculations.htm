Forecast calculations

 

 

 

 

Below is a simplified forecast diagram.

 

 

At every timestep, Precept:

 - checks if a new profile comes online or if a well is converted (see New profiles)

 - calculates the Production and Injection potentials for very wells using DCA curves (see Decline curves: Production and Injection potentials)

 - apply the necessary planned shutdowns (see Shutdowns)

 - apply the constraints to the production network

 

Forecast outputs are:

 - Constrained Production and injection at every nodes of the network (can be grouped by Project when visualizing results (see Forecast results: Charts)

 - shortfalls, grouped by Project

 - GHG emission

 - economics

 

 

Calculations

At every time step, Precept needs to solve a set of equations using Linear Solver.

 

Let's use the simple example below, where 3 wells flow only oil to a Node called "Manifold". A pipeline connects the node "Manifold" to a node called "FSO".

 

 

The following steps are performed at every timestep to calculate constrained forecast:

 

Step

Actions

 

 

 

 

1

 

Introduce the known 3 parameters  to the problem.

Known parameters are  Oil potentials for every wells.

Potentials are known since they derived from DCA curves.

Parameters are named:

 - Pot_Oil_Well_1

 - Pot_Oil_Well_2

 - Pot_Oil_Well_3

 

 

 

2

 

Introduce the unknown variables (Choke opening) to the problems.

 

In our example, there are 3 choke variables and are named:

-choke_Well_1

-choke_Well_2

-choke_Well_3

 

 

3

 

Introduce the unknown variables (oil flowing through pipeline) to the problems.

 

In our example, there is one pipeline from "Manifold" to "FSO".

Variable is named:

-Oil_from_Manifold_to_FSO

 

 

 

4

 

Formulate the conservation of Volume at every nodes: Whatever goes in = Whatever goes out

 

In our example, the conservation of volume is applied to "Manifold" nde only

 

Choke_Well1 * Pot_Oil_Well1  +  Choke_Well2 * Pot_Oil_Well2  +  Choke_Well3 * Pot_Oil_Well3  =  Oil_from_Manifold_to_FSO

 

 

 

5

 

Add Well constraints to the problem.

Choke openings need to be between 0 and 1.

 

  0 <=  Choke_Well1  <= 1

  0 <=  Choke_Well2  <= 1

  0 <=  Choke_Well3  <= 1

 

6

Add Node constraints to the problem, for example, if the FSO cannot process more than 100,000 bpd

 

Oil_from_Manifold_to_FSO <= 100,000

 

7

 

Add "Objective function" to the problem.

In our example, the objective is to maximize the variable  "Oil_from_Manifold_to_FSO"

 

8

 

Solve the Network and get the values of variables

 

 

9

 

Calculate constrained production
