probabilistic

 

 

 

 

Probabilistic can be generated by clicking on the "Probabilistic" icon in Precept Main User Interface.

 

The required inputs for Probabilistic forecast are:

 - Low (P90) / Mid (P50) / High Case (P10) potentials for every wells.

 - probabilistic distribution of duration for opportunities

 - Information regarding Wells Inter-dependency

 - number of runs to be generated

 

 

 

Low / Mid / High Case potentials

 Low (P90) / Mid (P50) / High Case (P10)  Cases potentials are generated using the DCA interface. (see Decline curves: Production and Injection potentials)

 

Check the "Probabilistic" option in "Settings" to visualize how the P10, P20, P30, ..., P90 will be generated from Low Mid and High cases.

 

 

Probabilistic distribution of duration

Probabilistic distribution of duration for opportunities can be generated by clicking on the "Probabilistic input" icon in Precept Main User Interface.

Users can then visualized all the Opportunities included in the scenario.

The expected duration has been previously entered when creating the Opportunities (see Opportunities)

For probabilistic forecast, Users then need to enter a min and max duration.

When generating probabilistic forecast, for every runs, the actual duration of opportunities will be generated using a triangular distribution.

 

 

Wells Inter-dependency

When generating probabilistic forecast, for every runs, by default, Precept will assign a potentials (P10, P20, ...  or P90) for every wells with equal probability.

 

Let's assume an oil field composed of 5 wells drilled in an homogeneous reservoir. Since the reservoir is homogeneous, one can consider that the P90 of full field is the sum of P90 of all individual wells.

If we let Precept assign P10, P20,..P90 to all wells individually, it is unlikely to have a run where all 5 wells are assigned the P90 profiles ( that probability is 1 / 9*9*9*9*9 ).

We therefore need to tell Precept that those wells are not independent, and if P90 is assigned to one well, P90 should be assigned to all the other wells.

 

That can be done by clicking the "Probabilistic input" icon in Precept Main User Interface, then select "Wells inter-dependency'
