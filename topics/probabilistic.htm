probabilistic

 

 

 

 

Probabilistic can be generated by clicking on the "Probabilistic" icon in Precept Main User Interface.

 

The required inputs for Probabilistic forecast are:

 - Low / Mid / High Case potentials for every wells.

 - probabilistic distribution of duration for oppotunities

 - number of runs to be generated, along with distribution of Low /Mid / High case.

 

 

 

Low / Mid / High Case potentials

Low / Mid / High Case potentials are generated using the DCA interface. (see Decline curves: Production and Injection potentials)

 

 

Probabilistic distribution of duration

Probabilistic distribution of duration for opportunities can be generated by clicking on the "Probabilistic input" in Precept Main User Interface.

Users can then visualizd all the Opportunities included in the scenario.

The expected duration has been previously entered in when creating the Opportunities (see Opportunities)

For probabilistic forecast, Users then need to enter a min and max duration.

When generating probabilistic forecast, for every runs, the actual duration of opportunities will be generated using a triangular distribution.

 

 

 

Probabilistic settings

Probabilistic settings needsto be entered by clicking on the "Settings" button in Precept Main User Interface.

In the example below, 200 runs will be run to generate the probabilistic forecast.

For each runs:

 - the Low case potentials will be used for !0% of the wells

 - the Mid case potentials will be used for 80% of the wells

 - the High case potentials will be used for 10%
