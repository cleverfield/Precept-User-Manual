Analogues

 

 

 

 

Data for analogue wells can be imported into Precept database and used as guidelines when generating DCA Curves for an asset.

 

To import analogue wells data, click on "Import Analogues data" from Precept Decline Curves modules.

 

Users can then import:

 

 Analogue Wells Meta data:

 - Asset name

 - well name

 - reservoir

 - Type (Oil Producer, Gas Producer, Water Injector , Gas Injector)

 - Unit for allocated production (SI or metric)

 - Coordinates

 

 

 Analogue Wells Production allocation data: (copy-paste from Excel)

 

 

Analogue Wells trajectories (Optional)

 

 

 

Users can then visualize historical production/injection and create DCA for analogue wells by selecting "Analogues" in the Decline Curves modules of Precept.
