Precept is installed using Microsoft ClickOnce to allow self-updates everytime the software is open.

 

License and config server

At opening, Precept needs to connect to the "license and config" server to check that:

 - User has a valid license

 - windows ID and password of the user are correct

 

 

Production Allocation database

Connection to production allocation database (Oil / water /gas / water Injection / Gas injection vs time) is needed for the Decline Curves Analysis module.

Production allocation data are also used as starting point for forecast.

It is recommended to have direct connection between Precept and production allocation database. However, production allocated data can also be uploaded and stored in Precept database.

 

Connection to existing production allocation database and creating the necessary queries is performed by Cleverfield during the installation / integration process.

 

 

Precept database

Precept database contains all the inputs needed to generate forecast (Asset model, Decline curves, schedule, Shutdowns, etc...).

Precept database also contains all the outputs of the forecast.

 

 

Communication port

Connection to Precept license and config server and Precept database are made through port 1433. This port needs to be open.

 

 

Connection from Third-party software

Connection from third-party software (Power BI, Spotfire, etc) to Precept database is possible as long as correct ID and password are used for the connection.
