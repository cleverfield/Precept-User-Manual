Patterns

 

 

 

 

Patterns property can be entered by clicking on the "Patterns Properties" icon in the Decline Curves Analysis (see Decline curves: Production and Injection potentials) module.

 

For Voidage and recovery calculations per patterns, the following inputs are needed:

 - STOOIP and GIP

 - Patterns coordinates (for visualization purposes)

 - Patterns allocation factors

 

 

 STOOIP and OGIP

Select "import Patterns Hydrocarbons in place" and fill in the required data. Copy-paste from Excel is supported.

 

 

Patterns Coordinates

Select "import Patterns coordinates" and fill in the required data. Copy-paste from Excel is supported.

On Field with only Producer wells, users will usually define one pattern for each wells. The well will be at the center of the pattern and the boundaries is the radius of investigation.

 

For a waterflood field with producers surrounded by injectors, users will usually define one pattern per producers. Boundaries will be defined using the surrounding injectors.

Patterns allocation factors

 

Pattern Allocation factors are used to sum the production and injection from different wells to calculate the voidage and recovery factor per patterns.

On the example below, the producer A1-W1 is surrounded by 2 Injectors A1-I1 and A1-I2.

 

The pattern named A1-W1 (name coming from the name of the producer) has been defined such that:

Pattern A1-W1 production  = 1* A1-W1 (well) production  + 0.5 * A1-I1 production + 0.5 * A1-I2 * production

Pattern A1-W1 injection = 1* A1-W1 (well) injection  + 0.5 * A1-I1 injection + 0.5 * A1-I2 * injection

 

 

Once all factors are entered, Users need to click on "Recalculate historical voidage" to calculate and saved in Precept database historical production and injection per patterns.
