Opportunities

 

 

 

 

The table below shows some examples of Opportunities, which are created to apply changes during forecast:

Most Opportunities will generate additional hydrocarbons production (drilling new well, stimulation, facilities upgrade). However, some opportunities will only implement shutdowns (ie: Maintenance) . Some opportunities like "Rig Move" are not really opporunity but needs to be implemented for accurate scheduling.

 

 

Example of Opportunity

Impact on forecast

Drilling new oil producer well

Introduce a new production profile

Drilling new Water Injector well

Introduce a new injection profile ,

but also incremental production profile to nearby producers

Work-Over

Resintall production of a well shut-in for integrity

Stimulation

Introduce a new incremental profile

Facilities upgrade

Modify a constraint at a specific node in the production Network

Maintenance

Shut a Node during forecast

Commissioning of new process facilities

Introduce a new node in production network

Lay down a new pipeline

Introduce a new pipeline and therefore a new routine in production network

Rig Move

Implemented for accurate scheduling, eventually introduce shutdown

 

 

A shutdown, a change in process constraint or introduction of a new profile could be implemented by setting manually a Start Date. However, Precept philosophy is to have the Opportunities part of a sequence of events, The Start and End Date of opportunities are therefore relative, which allows to generate What-if scenarios very easily (see Precept philosopy)

 

 

Add new Opportunities: Example 1: Drilling New Oil Producer

In the example below, an Opportunity named "A2-W4" is created

The duration of the opportunity (drilling) is 45 days.

 

The Start Date is set as "From sequence" which means there is not hard Start Date, it will be part of a sequence of events.

Color can be set for visualization purpose. This is the color used in the sequence of events.

 

That Opportunity can be executed anytime, therefore, the section "Earliest possible start date" is left blank

 

They are 4 different kind of impacts on forecast that can be introduced with an Opportunity:

 - New profiles

 - Shutdowns

 - Change in constraints.

 - Conversions

 

In this example,the new profile A2-W4, previously introduced (see section Add well to Asset Model) will be online once the opportunity is executed.

 

The cost of the opportunity (15 MM USD) will be used when generating forecast economics.

Add new Opportunities: Example 3: Compressor Upgrade

In the example below, an Opportunity named "B compr upgrade" is created.

The duration of the opportunity is 50 days.

The Start Date is set as "From sequence" which means there is no hard Start Date. The opportunity is  part of a sequence of events.

Color can be set for visualization purpose. This is the color used in the sequence of events.

 

The "B Compr upgrade" opportunity needs to be executed after the opportunity "B various maintenance" ends. If the Opportunity "B Compr upgrade" is placed before "B various maintenance" in the sequence of events, it will appear as "red".

 

The impact on Forecast is set to "Change constraint"

 

The Opportunity is part of the Project named  "Phase 2". The name of project is used when calculating and grouping shortfalls by projects.

 

The cost of the opportunity (20 MM USD) will be used when generating forecast economics.

 

Once the opportunity is executed, the node B_compr will have a new Total Gas constraint of 15,000 mscf/d (SI unit are used in this example therefore Gas unit is mscf/d.)

 

 

 

 

 

There is a shutdown associated with that Opportunity. Therefore the "impact on Forecast" also needs to be set to "shutdowns" to introduce the shutdown below.

 

The node B will be closed during the 50 days of execution.

 

 

 

Add new Opportunities: Example 4: Maintenance

In the example below, an Opportunity named "B various maintenance" is created.

The duration of the opportunity is 60 days.

 

The Start Date is set as "From sequence" which means there is no hard Start Date. The opportunity is part of a sequence of events.

 

 

Color can be set for visualization purpose. This is the color used in the sequence of events.

 

The "B Compr upgrade" opportunity can be executed anytime since "Earliest possible start date" inputs are left blank.
