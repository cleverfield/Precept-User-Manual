Security / Access policy

 

 

 

 

The security / Access policy can be set up by clicking on the "security" icon in Precept main User Interface.

 

Access to Asset

Access to an Asset (i.e User can see the Asset in the drop-down menu at Precept opening) is allowed when a User is added to the "authorized User".

Only "Super User" and the User who created the Asset can add new authorized User.

 

 

Security / Access Policies within an Asset

Within an Asset, additional security / Access policies can be set up to allow or not allow the option to:

 - create new scenario

 - edit a scenario created by other users

 - edit decline curves

 - view results
